# Nom du projet
Mini Blog Card

## Description
Carte de blog minimaliste (HTML + CSS) pour démonstration.

## Installation
1. Télécharger ou cloner le dossier `mini-blog-card`.
2. Ouvrir `index.html` dans un navigateur.

## Usage
- Modifier le contenu HTML (titre, description, auteur) dans `index.html`.
- Remplacer les images dans `resources/` si besoin.
